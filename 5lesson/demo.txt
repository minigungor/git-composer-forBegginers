Имеются git с графическими интерфейсами
gitsmart
github desktop и другие

разница лишь в интерфейсе программ

плюсы:
визуально подказывают изменения
реализуют командные интерфейсы которые есть в git

минусы:
некоторые ПО требуют лицензии

в smartgit - есть свои уникальные вещи
cherry-pick - когда мы копируем один из комитов на перед для изменений
revert - тоже самое делает, но откатывает все изменения которые были изначально в этом коммите
