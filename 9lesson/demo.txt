В случае если у нас появилась ошибка в коммитах, а еще она в неизвестной ветке

для этого используется git bisect
утилита в git которая спрашивает по поводу работы кода (хорошо, плохо) и параллельно
переключается на другие коммиты для проверки и нахождения ошибки

git bisect start 
git bisect good #hash
git bisect bad #hash

git bisect reset

в случае если мы хотим исправить коммиты
то можно воспользоваться git rebase -i #hash-до-которого-делать-rebase

откроется конфиг файл в котором будем писать команды которые будут делать по коммитам

edit
reword
pick

pick - оставить также
reword - переписать название коммита
edit - переписать файлы и тд
